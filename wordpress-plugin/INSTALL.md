# Sal Parts Voice AI - WordPress Plugin

## نسختان (Two Versions)

1. **salparts-voice-ai/** - نسخة عامة (General)
2. **salparts.com/** - نسخة جاهزة لـ salparts.com (Pre-configured for Sal Parts)

## التثبيت (Installation)

### لـ salparts.com:
1. ارفع مجلد `salparts.com` كـ ZIP
2. WordPress > Plugins > Add New > Upload
3. Activate
4. Settings > Voice AI - تأكد من الإعدادات وعدّل إن احتجت
5. زر المايكروفون سيظهر على الموقع

### للنسخة العامة:
1. ارفع مجلد `salparts-voice-ai` كـ ZIP
2. نفس الخطوات أعلاه
3. عدّل Store Name و Description في الإعدادات حسب موقعك

## الإعدادات (Settings)

- **App URL**: رابط تطبيق Voice AI المنشور (مثلاً على Vercel)
- **Store Name**: اسم المتجر
- **Store Description**: وصف المتجر - الـ AI يستخدمه لمعرفة ماذا تمثل
